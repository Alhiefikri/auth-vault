# Triage labels

## Label map

| Canonical role       | Label string       |
|----------------------|--------------------|
| Needs evaluation     | `needs-triage`     |
| Waiting on reporter  | `needs-info`       |
| Ready for AFK agent  | `ready-for-agent`  |
| Ready for human      | `ready-for-human`  |
| Won't fix            | `wontfix`          |

## Usage

The `triage` skill applies these labels when moving issues through the state machine. Always use the exact label strings above — do not create new labels for the same role.
