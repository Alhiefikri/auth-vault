# Issue tracker

## Type

GitHub Issues

## Repository

`Alhiefikri/auth-vault`

## CLI

Uses the `gh` CLI (`gh issue create`, `gh issue list`, `gh issue edit`, etc.).

## Workflow

- Creating an issue: `gh issue create --title "..." --body "..."`
- Listing open issues: `gh issue list`
- Adding a label: `gh issue edit <number> --add-label "<label>"`
- Removing a label: `gh issue edit <number> --remove-label "<label>"`
- Closing an issue: `gh issue close <number>`
